👨🏻‍💻 Customer Behavior Data Analysis Project

This project is an end-to-end data analytics portfolio project where I worked on analyzing customer behavior using real-world tools and techniques. It demonstrates my ability to take raw data, process it, and generate meaningful business insights.

📌 Project Overview

The goal of this project was to understand customer purchasing behavior and extract insights that can help businesses make better decisions.

In this project, I covered the complete data analysis workflow:

Data Cleaning & Exploration (Python)
Processed raw data, handled missing values, and performed exploratory data analysis to understand patterns.
Data Analysis (SQL)
Wrote SQL queries to analyze customer segments, purchasing trends, and key business metrics.
Data Visualization (Power BI)
Built an interactive dashboard to present insights in a clear and visual format.
Reporting
Summarized findings and provided actionable recommendations based on the analysis.
🛠️ Tools & Technologies Used
Python (Pandas, NumPy, Matplotlib/Seaborn)
SQL (MySQL/PostgreSQL/SQL Server)
Power BI
Jupyter Notebook
